# Emergency Service Directory Bangladesh



**One-click access to all emergency helplines in Bangladesh**  
Dial **999** or find specific services instantly — when every second counts.

[Live Demo](https://habib-web-dev1.github.io/emergency/) • [Repository](https://github.com/habib-web-dev1/emergency)



</div>

## Features

- National Emergency: **999** (Police • Fire • Ambulance)
- Categorized helplines (Police, Fire, Health, Women & Child, Utilities, etc.)
- One-tap **Copy number** functionality
- Fully responsive — works perfectly on mobile & desktop
- Fast loading • No tracking • Offline-friendly
- Clean, easy-to-read design with icons

## How to Use

1. Visit → [habib-web-dev1.github.io/emergency](https://habib-web-dev1.github.io/emergency/)
2. Scroll or search for the service you need
3. Tap **"Copy"** button → number copied!
4. Paste & call immediately

In any emergency: Just dial **999**
